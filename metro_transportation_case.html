Case Study 1: Facility Placement in Aryavarta

Problem Statement

Aryavarta faces challenges in ensuring equitable access to essential services like healthcare and public safety. The task is to determine the optimal placement of facilities such as hospitals and police stations using efficient algorithms.

Algorithms Used

1. Floyd-Warshall Algorithm

  • Purpose: Find the shortest paths between all pairs of locations in the city.
  • Use: Helps identify central points for facility placement to minimize travel time.
  • Impact: Reduces response times in emergencies.
View Floyd-Warshall Code

2. AVL Tree

  • Purpose: Manage population data efficiently.
  • Use: Stores and retrieves population densities to prioritize areas for facilities.
  • Impact: Enables dynamic updates and efficient decision-making.
View AVL Tree Code

3. Sorting

  • Purpose: Rank locations based on priority factors such as population density.
  • Use: Ensures high-priority areas are considered first for facility placement.
  • Impact: Optimizes resource allocation.
View Sorting Algorithm Code

4. Depth-First Search (DFS)

  • Purpose: Ensure connectivity of the city graph.
  • Use: Verifies that no area is isolated from essential services.
  • Impact: Promotes equitable access across the city.
View DFS Algorithm Code

Results

The application of these algorithms identified the following facility locations:

Facility Locations
Hospitals Sector 5, Sector 12, Sector 20
Police Stations Sector 3, Sector 15, Sector 27
