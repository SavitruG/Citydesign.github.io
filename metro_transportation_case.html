Case Study 1: Metro Public Transportation

By Kushal Koujalagi

Overview

Aryavarta is a growing city. Traffic is becoming a big problem because there are too many vehicles on the road. A metro system is being planned to connect important areas like Central Square, Tech Park, City Mall, Education Hub, Industrial Zone, and Residential Area. This will help people travel faster and reduce traffic on the roads.

Algorithms Used

Dijkstra's Algorithm

This algorithm helps us find the shortest path between two stations.

Kruskal's Algorithm

Used to design the metro network with the lowest cost.

Bellman-Ford Algorithm

Finds the shortest path while considering delays or extra travel time.
