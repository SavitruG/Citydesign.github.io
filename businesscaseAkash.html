PROBLEMS AND SOLUTIONS

Optimized Problems and Solutions for Travel and Tourism Management

1. Sorting Travel Packages by Cost

Problem: Sorting multiple travel packages by price to find the most affordable package for tourists. Example prices (converted to INR at 1 USD = 80 INR):

Package Price (INR)
Package A ₹120,000
Package B ₹96,000
Package C ₹144,000
Package D ₹104,000
Package E ₹80,000

Solution: Use the Merge Sort algorithm, which efficiently sorts the travel packages by price in O(n log n) time.

After sorting the packages by price:

Package Price (INR)
Package E ₹80,000
Package B ₹96,000
Package D ₹104,000
Package A ₹120,000
Package C ₹144,000

2. Sorting Hotels by Rating

Problem: Sorting hotels by customer ratings to prioritize the highest-rated hotels. Example ratings:

Hotel Rating (1-5)
Hotel A 4.5
Hotel B 3.8
Hotel C 5.0
Hotel D 4.2
Hotel E 4.7

Solution: Use the Heap Sort algorithm to efficiently sort the hotels by rating in O(n log n) time.

After applying heap sort, the list becomes:

Hotel Rating (1-5)
Hotel C 5.0
Hotel E 4.7
Hotel A 4.5
Hotel D 4.2
Hotel B 3.8

3. Finding the Shortest Path for City Tours

Problem: Finding the shortest route for a city tour. Distances between locations:

From Location To Location Distance (km)
A B 2
A C 5
B C 3
B D 4
C D 1

Solution: Use Dijkstra’s Algorithm to find the shortest route. The optimal route is:

A -> B (2 km) -> B -> C (3 km) -> C -> D (1 km) Total distance: 6 km

4. Allocating Resources for Sustainable Tourism

Problem: Allocating resources for eco-friendly transport, green hotels, and clean energy. Example allocation (converted to INR):

Resource Type Allocation (INR)
Eco Transport ₹24,00,000
Green Hotels ₹40,00,000
Clean Energy ₹16,00,000

Solution: Use Kruskal’s Algorithm to allocate resources efficiently, ensuring minimal costs while achieving sustainability goals.

5. Scheduling Travel Agents for Tours

Problem: Scheduling travel agents for various tours based on their availability. Example agent data:

Agent Tour Type Availability (days)
Agent A City Tour 3
Agent B Nature Tour 5
Agent C Cultural Tour 2
Agent D City Tour 4

Solution: Use Heapify (Heap Sort) to allocate agents based on their availability. After sorting by availability:

Agent Tour Type Availability (days)
Agent C Cultural Tour 2
Agent A City Tour 3
Agent D City Tour 4
Agent B Nature Tour 5

Problem: Allocating a travel budget for various categories. Example budget distribution (converted to INR):

Category Amount (INR)
Accommodation ₹30,000
